# 📸 教科書クイズ生成アプリ

スマホで教科書の写真を撮るだけで、自動的に小テスト問題を生成するWebアプリです。

## ✨ 主な機能

### 🎯 コア機能
- **📸 写真から自動問題生成** - 教科書・ノート・プリントの写真をアップロード
- **🤖 AI画像認識** - Google Gemini Vision APIで高精度なテキスト抽出
- **📝 4種類の問題タイプ**
  - 単語問題（4択）
  - 語順並べ替え
  - 翻訳問題
  - 読み当て問題
- **✅ 自動採点** - AI採点で部分点にも対応

### 💡 特徴
- ✅ **無料で使える** - Google Gemini APIの無料枠内で利用可能
- ✅ **APIキーをローカル保存** - プライバシー保護（サーバーには送信しない）
- ✅ **スマホ対応** - レスポンシブデザイン
- ✅ **オフライン問題生成** - 画像認識以外はオフラインで動作

## 🚀 使い方

### 1. セットアップ
1. アプリにアクセス
2. [Google AI Studio](https://makersuite.google.com/app/apikey) からGemini APIキーを無料で取得
3. アプリ上部のAPIキー設定エリアにAPIキーを入力して保存

### 2. 問題作成の流れ
1. **📸 写真撮影** - スマホカメラで教科書を撮影
2. **📤 アップロード** - 画像ファイルを選択またはURL入力
3. **⏳ AI解析** - 自動でテキスト抽出（10-30秒）
4. **📝 問題タイプ選択** - 4種類から選択
5. **✍️ 解答** - 問題を解いて自動採点！

## 📝 注意事項

### Gemini APIキーについて
- APIキーは**ブラウザのlocalStorageに保存**されます
- サーバーには送信されません
- 無料枠: 1分あたり15リクエスト、1日1500リクエスト
- [料金詳細](https://ai.google.dev/pricing)

### 画像認識の精度
- **クリアな画像**ほど精度が高い
- 推奨: 明るい場所で正面から撮影
- 手書き文字よりも印刷文字の方が認識しやすい

## 🔒 プライバシー

- **APIキー**: ブラウザのローカルストレージに保存（サーバーには送信しない）
- **画像データ**: Google Gemini APIに送信（処理後は保存されない）
- **問題データ**: すべてクライアント側で処理
